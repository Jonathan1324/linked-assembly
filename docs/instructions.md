# Supported Instructions

## x86

### 16-bit

> Not supported yet. *(planned)*

### 32-bit

#### Control Flow & CPU

- `nop`
- `ret`
- `ret imm16`
- `int3`
- `int1`
- `hlt`
- `pause`
- `wait` / `fwait`
- `int imm8`

#### Flags & Interrupts

- `cli` / `sti`
- `clc` / `stc` / `cmc`
- `std`

#### CPU Info

- `cpuid`
- `rdtsc`

#### MOV

- `mov reg, imm`
- `mov reg, reg`

#### Stack

- `push reg`
- `push imm`
- `pop reg`
- `pusha`/`popa`
- `pushf`/`popf`

### 64-bit

> Not supported yet. *(planned)*

## ARM

> Not supported yet.

## RISC-V

> Not supported yet.
