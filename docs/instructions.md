# Supported Instructions

## x86

### 16-bit
> Not supported yet. *(planned)*

### 32-bit

#### Control Flow & CPU
- `nop`
- `ret`
- `ret imm16`
- `int3`
- `int1`
- `hlt`
- `pause`
- `wait` / `fwait`

#### Flags & Interrupts
- `cli` / `sti`
- `clc` / `stc` / `cmc`
- `std`

#### CPU Info
- `cpuid`
- `rdtsc`

#### MOV
- `mov reg, imm32`
- `mov reg, reg`

### 64-bit
> Not supported yet. *(planned)*

## ARM
> Not supported yet.

## RISC-V
> Not supported yet.
